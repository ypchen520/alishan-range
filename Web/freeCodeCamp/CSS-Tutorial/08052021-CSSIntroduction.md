## CSS Selectors

* Element selector
  * the name of the tag
  
  ```CSS
  p {

  }
  ```

* Class selector
  * dot

  ```CSS
  .aclass {
      
  }
  ```

  ```HTML
  <h2 class="aclass"></h2>
  ```

* ID selector
  * ```#``` (a hash symbol)
